To execute the Postman project :

Download the Petstore Automation_Postman.postman_collection and Petstore_GlobalVariable.postman_environment from GitHub
Import the project in Postman tool (Open source)
Click on the arrow in the right side and choose RUN option.
Select Environment file which is shared and click on Run.
If everything is Green, testing is successful.

Test cases are created in a flow based. 
Values are parameterized to fetch the data during run-time.
Assertions are added to verify the response fields and status codes.
