To execute the SoapUI project :

Download the Petstore project from GitHub
Import the project in SoapUI (Open source)
Expand the project and "Flow" test suite
Double click on "Postive Flow" test case and click on green arrow to execute.
If everything is Green, testing is successful.

Test cases are created in a flow based. 
Values are parameterized to fetch the data during run-time.
Assertions are added to verify the response fields and status codes.
